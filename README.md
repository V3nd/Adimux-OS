# Adimux OS

Adimux OS v1.0: KSPP-hardened LFS 12.4 distro with Rust kernel modules.
Minimal attack surface for secure personal/server use.
Automated builds + CVE patching.

## Layout
- kernel/   : Linux source (submodule)
- configs/  : kernel configs
- patches/  : patches (incl. CVE fixes)
- scripts/  : build automation
- docs/     : documentation
